# Lagrangian Ice Tracking System 

All of the functions for LITS are in:
- LITS_age.py

You can run the model from:
- run_LITS.py

The maps.py file contains some functions to plot correctly the tracers. 

This code uses the SIV and SIC v4 data from NSIDC on the 25km EASE grid. The 
data was not included in this repo for data management through github. 

There is also a bunch of tracers and data variables specifically for the Beaufort Sea. 
My project at the time was focusing on it. 

LITS also uses matlab (specifically for the interpolation of the velocity tothe grid), 
so you need it installed on your computer for this specific version
